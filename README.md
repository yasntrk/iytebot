### REQUIREMENTS

- Python
- Flask
- A .pt Chatbot GPT-2 MODEL

### BUILD

```bash
pip install -r requirements.txt
```
You must train a GPT-2 model and place it along with the config.json file inside the content directory.

## RUNNING THE APPLICATION

```bash
python app.py
```
